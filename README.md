## Period

2022.02 ~ 2022.04

## Content

- 메인 TabView에 영단어 리스트, 영단어 카드, 미니게임 뷰 등으로 이루어져 있으며 SwiftUI로 구현.
- Firebase의 RealTime Database를 이용해 영단어 리스트뷰 개발.
- onTapGesture, DragGesture를 이용한 영단어 카드 뷰 기능 개발.
- Firebase DB에서 랜덤으로 불러온 영단어를 이용해 사천성 게임 기능 개발.
